# Feature Guide - ResumeAi

Welcome to ResumeAi! This document explains the core capabilities of the platform.

## 1. Deep Resume Analysis
- **ATS Scoring**: Get a realistic score of how your resume performs against modern Applicant Tracking Systems.
- **JD Matching**: Paste a job description and see exactly which keywords you are missing.
- **AI Improvement Suggestions**: Receive personalized feedback on how to rewrite sections of your resume for better impact.

## 2. Dynamic Resume Generation
- **Templating**: Choose from professionally designed templates.
- **Auto-Fill**: Integration with location and education databases makes filling out the resume builder fast and accurate.
- **Export to PDF**: Generate clean, ready-to-use PDFs in seconds.

## 3. Smart Cover Letters
- **Tailored Content**: Generate cover letters that contextually link your experience to the specific job you're applying for.
- **Review & Edit**: Refine the AI-generated letter before finalizing.

## 4. Integrated Emailing
- Send your optimized resume and cover letter directly to recruiters from within the app.

## 5. Administrative Controls (Admin Only)
- **User Monitoring**: View all registered users and their current usage status.
- **Usage Quotas**: Admins can increase or decrease analysis/generation limits for specific users.
- **Role Management**: Easily promote users to Admin or demote them to standard User roles.
- **Upgrade Workflow**: Process user requests for higher usage limits via a dedicated approval dashboard.

## 6. Security & Privacy
- **JWT Authentication**: Secure, stateless login.
- **Data Protection**: Sensitive information (like passwords) is handled with industry-standard encryption and never exposed via APIs.
